# RPG Maker MZ CoreScript - Core Modules

## Core Components (`rmmz_core`)

These modules form the foundation of the RPG Maker MZ engine:

- **Bitmap.js**: Handles image rendering and manipulation
- **ColorFilter.js**: Provides color filtering capabilities for visual effects
- **Core.js**: Contains fundamental engine definitions and initialization
- **Graphics.js**: Controls the rendering pipeline and graphical presentation
- **Input.js**: Manages keyboard and controller input
- **JsonEx.js**: Extensions for JSON handling, particularly for game data saving/loading
- **Point.js**: Simple coordinate system implementation
- **Rectangle.js**: Rectangle geometry for UI and collision detection
- **ScreenSprite.js**: Special sprite for screen-level effects
- **Sprite.js**: Base visual element for game objects
- **Stage.js**: Container for visual elements
- **Tilemap.js**: Handles map tile rendering and management
- **TilingSprite.js**: Sprite that can be tiled/repeated
- **TouchInput.js**: Touch and mouse input handling
- **Utils.js**: Utility functions used throughout the engine
- **Video.js**: Video playback capabilities
- **Weather.js**: Weather effect system
- **WebAudio.js**: Audio playback and management
- **Window.js**: Base window implementation for UI elements
- **WindowLayer.js**: Layer management for windows
